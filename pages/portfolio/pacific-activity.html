Pacific Activity

In this activity, i've learn how to embed html elements and also applying styles. their Basic tags were the first lessons. Creating headings, paragraphs, and links seemed straightforward, but the real learning came in understanding the importance of semantic HTML. Using appropriate tags not only makes the content more accessible to users, especially those with disabilities, but also improves SEO, making it easier for search engines to index the content. My first encounter with CSS was both intriguing and overwhelming. Understanding the separation of structure (HTML) and presentation (CSS) was a fundamental shift in thinking about web development. The idea that CSS could control the look and feel of a website independently from the HTML content opened up vast creative possibilities. However, the learning curve was steep, with numerous properties and values to comprehend and remember.
